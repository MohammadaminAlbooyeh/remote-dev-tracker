# remote-dev-tracker

> A time tracking system for remote developers — Clock In/Out, CET timezone, hourly rate calculation, admin dashboard, and CSV export.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Backend** | FastAPI, Python |
| **Database** | PostgreSQL |
| **Auth** | JWT (python-jose) |
| **Password** | bcrypt |
| **Timezone** | pytz (Europe/Rome) |
| **CSV** | Python csv module |
| **Frontend** | React |
| **HTTP Client** | Axios |
| **Deployment** | Docker + docker-compose |

**GitHub:** `https://github.com/MohammadaminAlbooyeh/remote-dev-tracker`
